Multi-Device Sync

Your data lives in this browser by default. To see the same invoices & clients on every computer, link a single JSON file inside a cloud folder (Dropbox, iCloud Drive, OneDrive, or Google Drive desktop app). Every save writes to that file; other computers load from it.

Status: Not linked — data is local only

Setup (once per computer): Put the file in a cloud-synced folder → click Link / Create Sync File → choose that file (or save a new invoiceflow-data.json into the folder). Use Chrome or Edge for best results (file link is remembered). On Safari, use Export on one Mac and Import on the other, or place the file in iCloud and Import it.
